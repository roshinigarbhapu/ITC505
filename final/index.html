Addendum

The process is it's a grid-based puzzle game where the goal is to turn off all the lights by toggling them. Each toggle affects adjacent lights.

The creative choices of "Lights Out" game are visual design, user interface and grid layout.

One challenge is designing an efficient algorithm to toggle lights and check for win conditions, especially as the grid size increases and ensuring the game working well.
